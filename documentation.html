GET/order/:id
Get order by id

Sample Query:

GET https://pacific-reef-95638.herokuapp.com/order/1
PUT/order/:id
Update order status

Order status can be one of: Completed, Pending, Cancelled

POST https://pacific-reef-95638.herokuapp.com/order/1
CONTENT-TYPE: application/json
ACCEPT: application/json

{
  "status": "Completed"
}
