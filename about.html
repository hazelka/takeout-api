About Restaurant Management API

What's the purpose behind this API?

The Restaurant Management API allows access and update of customer orders, inquiry messages and menu items. It is the backend of the full stack application Takeout Management System.

What did you build the API with?

This is a web application hosted on Heroku. The tech stack is listed below:

  • Node.js
  • Express.js
  • PostgreSQL
  • WebSocket
